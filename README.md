# Headless Day

A collection of [Wix Headless](https://www.wix.com/studio/developers/headless) projects built by Wix employees during our internal **Headless Day** — July 9, 2026.

## About

Headless Day is an internal event where Wix employees build projects on top of Wix Headless. This repository is the shared home for those projects.

## Contributing

Wix employees participating in Headless Day: add your project as a subdirectory at the root of this repo.

- Create a folder named after your project (e.g. `my-cool-app/`).
- Include a short `README.md` inside your project folder describing what it does and how to run it.
- Open a pull request to merge your project.
