# Vechornytsi — design system

Ukrainian editorial restraint meets Nordic-spare warmth. One table, no clutter,
every element intentional. Negative space carries the composition; a single
photograph commands each section. High-contrast Cormorant Garamond display
against clean Hanken Grotesk body.

- **paper** `#F5F0EB` — warm off-white page canvas
- **paper-warm** `#FFFDF9` — cards, tiles, raised surfaces
- **ink** `#1A1F24` — deep timber near-black; body text and the dark palate-cleanser section
- **ink-soft** `#2C3E50` — brand slate; primary actions and headings emphasis
- **mute** `#5A5550` — secondary/muted text
- **rule** `#D6CFC6` — hairline borders and dividers
- **accent** `#A3B18A` — sage; kickers, small caps, left-border accents
- **cream** `#F5F0EB` — off-white text on dark sections

Mood: intimate · Nordic · refined · elemental. Motion is CSS-first and minimal —
slow reveals, quiet opacity fades, honored `prefers-reduced-motion`.
